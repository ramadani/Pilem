# Pilem

Latihan membuat app android menggunakan API TMDb
* Top Rated Movies
* Popular Movies
* Upcoming Movies
* Now Playing

> Done is better than perfect

### Roadmap
* [x] Explore & test REST API TMDb
* [x] Buat app base design (warna dan atribut lainnya dari TMDb)
* [x] Implement Model View Presenter (MVP)
* [x] Lazy load image
* [x] Endless Scrolling
* [ ] Swipe Refresh
* [ ] Tab Layout & Fragment
